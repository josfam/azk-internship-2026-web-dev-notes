## 1. Make sure you have key software installed (frontend)

- [ ] Install nodejs (to run Javascript on your computer)
	- Surprisingly non-trivial to do so from the official site (https://nodejs.org/en/download)
	- It is somewhat easier to do so via `nvm` (node version manager) at (https://www.nvmnode.com/guide/download.html)
- [ ] Install pnpm (to do smarter dependency management for your projects)
	- pnpm (https://pnpm.io/) is installed via node's own npm (node package manager)
	```sh
	npm install -g pnpm
	```
- [ ] Install python (to run python on your computer)
- [ ] A modern IDE (like VS Code, and any other that you prefer -- but make sure that you also)
